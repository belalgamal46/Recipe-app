# 📖 Blog App <a name="about-project"></a>

**The Recipe** app manages recipes, ingredients, and inventory. Save ingredients, track what you have, create recipes, and generate a shopping list. Share recipes publicly for easy access.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

You will need the following:
- A code editor for example: Rubymine, Vscode
- Ruby installed on your pc
- Install rails on your pc you can use the following command to install it:

    ```
    gem install rails
    ```
- You will need PostgreSQL to be installed on your PC to handle the database if you are running the project locally, or you can use Docker for easier implementation.

### Setup

Clone this repository to your desired folder:

```
  git clone https://github.com/belalgamal46/Recipe-app.git
```

```
  cd Recipe-app
```

### Install

Install this project with:

```
  gem install
```

**OR**

```
  bundle install
```

### Usage

To run the project, execute the following command:

```
  rails server
```
